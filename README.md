 🥋 WebDojo — Prática de Automação com Cypress

Repositório utilizado para praticar automação de testes com Cypress 
durante a certificação **Ninja do Cypress**.

## 🛠️ Tecnologias

- Cypress
- JavaScript
- Node.js

## 🧪 Testes desenvolvidos

- **Login** — validação de autenticação
- **Consultoria** — preenchimento e validação de formulário
- **Commands** — criação de comandos customizados no Cypress
